SmartChurn: Targeted Discounts for Customer Retention.

Executive Summary

Customer churn is a major threat to profitability. In this project, we build a machine learning model to predict churn, identify key drivers, and simulate the impact of targeted 20% discounts on high-risk customers.

Goal: actionable strategies to improve retention and profitability.

Random Forest Model

We apply a Random Forest classifier with stratified 5-fold cross-validation. Random Forest is chosen because it can capture non-linear relationships, handle noisy data, and provides interpretable feature importance.

In[9]:
from sklearn.model_selection import StratifiedKFold
from sklearn.metrics import accuracy_score, precision_score, recall_score
from sklearn.ensemble import RandomForestClassifier

# Initialization
pred_train_labels = np.zeros(shape=(X.shape[0], 2))
feature_importance_df = pd.DataFrame({
    'feature_name': X.columns,
    'feature_importance': [0]*len(X.columns)})

# Stratified K-Fold CV
kfold = StratifiedKFold(n_splits=5, shuffle=True, random_state=29)
fold_counter = 1

for train_index, test_index in kfold.split(X, y):
    X_train, X_test = X.iloc[train_index], X.iloc[test_index]
    y_train, y_test = y[train_index], y[test_index]
    # Build model
    rf = RandomForestClassifier(random_state=56)
    # Train model
    rf.fit(X_train, y_train)
    # Predict
    pred_train_labels[test_index] = rf.predict_proba(X_test)
   # Accumulate feature importance
    feature_importance_df['feature_importance'] += rf.feature_importances_
    
    print(f"Fold {fold_counter} Precision {precision_score(y_test, rf.predict(X_test)):.3f} "
          f"Recall {recall_score(y_test, rf.predict(X_test)):.3f} "
          f"Accuracy {accuracy_score(y_test, rf.predict(X_test)):.3f}")
    fold_counter += 1

# Overall metrics
pred_y = pred_train_labels.argmax(axis=1)
print("\nTotal Metrics")
print(f"Total  Precision {precision_score(y, pred_y):.3f} "
      f"Recall {recall_score(y, pred_y):.3f} "
      f"Accuracy {accuracy_score(y, pred_y):.3f}")

Simulate 20% Discount Scenario

Now comes the business challenge: can offering a 20% discount reduce churn and increase profit?

  • Intuition: Price changes alone are unlikely to fully prevent churn, since our earlier analysis showed weak correlation between pricing and customer retention.
  • Approach: To test this, we simulate a 20% discount scenario and calculate the expected profit for each customer, both before and after applying the discount.
  • Formula: ExpectedProfit = ProbabilityofStaying CustomerProfit

The results show that while some customers indeed have a lower churn probability with a discount, the benefit must be carefully weighed against the loss of revenue caused by reduced prices.

# Expected profit calculation
expected_no_dis, expected_dis = [], []

for idx, row in df.iterrows():
    cid = row['id']
    
    # Profit Without Discount
    profit_no_dis = row['cons_12m']*row['var_year_price_off_peak_var'] + row['cons_gas_12m']*row['var_year_price_off_peak_fix']
    prob_stay = customer_prob.loc[customer_prob['id']==cid,'0'].values[0]
    expected_no_dis.append(prob_stay * profit_no_dis)

    # Profit With 20% Discount
    profit_dis = row['cons_12m']*row['var_year_price_off_peak_var']*0.8 + row['cons_gas_12m']*row['var_year_price_off_peak_fix']*0.8
    prob_stay_dis = customer_prob.loc[customer_prob['id']==cid,'0_dis'].values[0]
    expected_dis.append(prob_stay_dis * profit_dis)

# Assign to DataFrame
customer_prob['expected_no_dis'] = expected_no_dis
customer_prob['expected_dis'] = expected_dis
# Difference between discounted and non-discounted expected profit
customer_prob['diff_discount'] = customer_prob['expected_dis'] - customer_prob['expected_no_dis']

Final Recommendation

The analysis shows that while discounts alone do not guarantee churn prevention, selective discounting can still be financially beneficial. Instead of offering a blanket discount to all customers, the company should target high-risk churners where the expected profit gain outweighs the revenue loss.
